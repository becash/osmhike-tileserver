Osmhike

Hiking oriented Tile Server

Osmhike is a Linux/Docker based implementation of an Openstreetmap Tile Server. It uses Openstreetmap data.
It is derived from Cyclosm.

Three goals:

  • Allow to build a personnal tile-server, for producing maps adapted to mountain hiking, (without loosing too much bike information)
  • Provide a full beginner's tutorial ( the one I would have liked to have at the beginning of the project )
  • Implement contours and hillshade, which are missing in Openstreetmap
